# AtmosStore

AtmosStore는 분위기에 따라 장소를 소개하는 정적 웹사이트입니다.  
HTML과 CSS를 사용하여 메인 페이지와 분위기별 페이지를 구현했습니다.

본 프로젝트는 2025년부터 인공지능소프트웨어학과에 재학하며 약 4개월간 학교 수업에서 배운 내용을 바탕으로 외부 도움 없이 개인적으로 구현한 학습용 프로젝트입니다.  

완성도보다는 HTML과 CSS의 기본 구조 이해와 레이아웃 및 스타일링 구현 경험에 중점을 두었습니다.

## 페이지 구성

- 메인 페이지: Cozy / Hip 분위기 선택
- Cozy 페이지: 편안한 분위기의 장소 소개
- Hip 페이지: 트렌디한 분위기의 장소 소개

## 주요 기능

- 분위기별 페이지 분리
- 이미지와 텍스트를 활용한 장소 소개
- CSS를 이용한 레이아웃 및 스타일링
- 각 분위기에 맞는 색상과 디자인 적용

## 사용 기술

- HTML5
- CSS3

## 프로젝트 목적

본 프로젝트는 웹 개발 기초 학습을 목적으로 제작되었습니다.  
정적 웹사이트 구조와 CSS를 활용한 화면 배치 및 디자인 구현을 연습하기 위해 진행했습니다.
